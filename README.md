# NumPy Practices for Data Science

A comprehensive hands-on notebook covering the fundamentals and essential concepts of **NumPy**, the core numerical computing library in Python. This project was created as part of my Data Science learning journey to strengthen my understanding of array operations, mathematical computations, and data manipulation.

## 📌 Project Overview

This notebook demonstrates practical examples of NumPy concepts, starting from basic array creation to advanced indexing and mathematical operations. Each section includes simple examples to help understand how NumPy works in real-world data science workflows.

## 📚 Topics Covered

* Introduction to NumPy
* Creating NumPy Arrays
* Array Attributes

  * Shape
  * Size
  * Dimensions
  * Data Types
* Array Indexing
* Array Slicing
* Boolean Indexing
* Fancy Indexing
* Array Reshaping
* Flattening Arrays
* Mathematical Operations
* Statistical Functions
* Aggregation Functions
* Broadcasting
* Universal Functions (ufuncs)
* Sorting Arrays
* Searching Elements
* Copy vs View
* Deep Copy vs Shallow Copy
* Random Number Generation
* Matrix Operations
* Useful NumPy Functions
* Practice Examples

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/NumPy-Practices.git
```

2. Navigate to the project folder

```bash
cd NumPy-Practices
```

3. Install NumPy (if not already installed)

```bash
pip install numpy
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open **Numpy_Practices.ipynb** and run the cells.
